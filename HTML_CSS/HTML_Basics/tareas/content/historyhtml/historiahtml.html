HTML

  • HTML son las siglas de Hipper Text Markup Language, que se podría traducir como lenguaje de marcado de hipertexto.

  • El lenguaje HTML se utiliza para crear páginas web. Es muy sencillo ver el código HTML de una página web, la opción para visualizarlo cambia de un navegador a otro, y también puede variar la versión en un mismo navegador, pero suelen tener opciones similares.

  • HTML está formado por etiquetas que se ecriben entre simbolos de mayor que y menor que respectivamente.

Historia de HTML

  • Los principios del lenguaje HTML vienen del año 1991, cuando Tim Berners-Lee creo la primera página web.

Versiones de HTML

  • Tim Berners-Lee definio la primera versión de HTML en el año 1991.

  • En la actualidad la última versión de HTML es HTML 5.
